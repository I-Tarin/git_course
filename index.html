Команды для Git в консоль

  • "git config --global user.name "My Name" - имя пользователя
  • "git config --global user.email myEmail@example.com" - адрес электронной почты
  • "git config --list" - посмотреть все настройки системы
  • "git status" - статус состояния
  • "git add < file >" - включение слежения за файлом
  • "git rm --cached < file >..." - отключение слежения за файлом
  • "git add ." - слежение\индекс за всеми файлами проекта
  • "git commit -m "edit html"" - коммит файлов проекта, в кавычках кратко суть
  • "git branch" - состояние ветвления
  • "git branch test" - создание ветвления, с веткой test
  • "git branch -D test" - удаление ветвления, ветки test
  • "git checkout readme" - переход на ветку → readme
  • "git checkout -b new" - создание и переход на ветку → new
  • "git merge test" - слияние веток master и test (выполняется с корневой ветки master)
  • "git remote add origin https://github.com/I-Tarin/git_course.git" - подключение локального репозитория к GitHub
  • "git push -u origin master" - отправка репозитория на сервер GitHub
  • "git push" - отправка изменений на сервер GitHub
